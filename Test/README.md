RGV简介：
=======
    RGV，是有轨制导车辆（Rail Guided Vehicle）的英文缩写 ，又叫有轨穿梭小车， RGV小车可用于各类高密度储存方式的仓库，
    小车通道可设计任意长,可提高整个仓库储存量，并且在操作时无需叉车驶入巷道，使其安全性会更高。在利用叉车无需进入巷道的
    优势，配合小车在巷道中的快速运行，有效提高仓库的运行效率。
